# ☕ Brewed in Lahore

A modern, responsive café website built with React and Tailwind CSS. Designed as a real-world frontend portfolio project showcasing multi-page layout, component architecture, and clean UI design.

**Live Demo:** [https://bbrewed.netlify.app/]  
**Built by:** Hammd — [github.com/Hammd04](https://github.com/Hammd04)

---

## Pages

- **Home** — Hero, About, Services, Menu preview, Testimonials
- **Menu** — Full menu with Hot and Cold Coffee categories
- **About** — Brand story, values, and stats
- **Contact** — Contact form, opening hours, location map

---

## Tech Stack

- React 18 (Vite)
- Tailwind CSS v4
- React Router DOM

---

## Features

- Fully responsive — mobile, tablet, desktop
- Sticky navbar with scroll-based color change
- Multi-page routing with React Router
- Custom color palette and Google Fonts
- Real Pakistani context — Lahore address, PKR pricing, local copy

---

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
